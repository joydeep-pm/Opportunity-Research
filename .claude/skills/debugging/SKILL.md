# Skill: Debugging

## Goal
Find root cause quickly, verify fix, and prevent recurrence.

## Workflow
1. Reproduce reliably.
2. Isolate failing layer (UI, orchestration, API, persistence).
3. Validate root cause with evidence (logs/tests/trace).
4. Implement narrow fix.
5. Add or run regression checks.

## Debug Priorities
- Crashes and broken CTAs.
- Incorrect routing/state transitions.
- API response contract mismatch.
- Storage/SSR boundary issues.
